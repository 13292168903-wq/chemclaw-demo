## Key Architectural Insight
The integration works by using BOTH OpenClaw and local parser:
- **OpenClaw** = the AI brain (Qwen3_6 model, Skill knowledge, conversational explanations)
- **Local parser** = the structured data extractor (charts, 3D coordinates, metrics, quiz)
This hybrid approach means the Web Demo gets the best of both worlds:
OpenClaw's intelligence + structured visualization that OpenClaw can't provide
